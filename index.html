About

I focus on AI safety, specifically in model evaluations and Reinforcement Learning from Human Feedback (RLHF). My work is driven by the challenge of aligning AI systems with human values and user needs, with a focus on optimizing these systems for long-term goals that benefit individuals and organizations. I have a strong interest in eliciting preferences from both models and users to enhance security and improve performance in human-AI interaction.

Though my work focuses on AI safety, my background in computational neuroscience has shaped my approach to aligning sensor data with model embeddings. I developed methods for reconstructing speech from neural data and aligning artificial neural network embeddings with brain activity, enhancing understanding of how biological neural networks process speech.

My goal is to contribute to optimizing AI in service of the long-term objectives of its users, promoting security and scientific progress.

Recent Projects

Intracranial Speech Processing [View Repository]

  • Developed an end-to-end pipeline for analyzing intracranial neural data aligned with audio and language features, supporting preprocessing, feature extraction, and anatomical localization.
  • Extracted audio and language representations (e.g., GPT-2 embeddings) from speech tasks, enabling high-resolution analysis of neural responses to auditory and linguistic inputs.
  • Applied methods to correlate time-resolved neural activity (1000 Hz sample rate) with annotations of auditory and language stimuli, identifying feature-responsive electrodes and their anatomical dynamics.

GitHub Repository Structure Analyzer [View Repository]

  • Built a tool to extract, analyze, and evaluate GitHub repository structures, generating JSON outputs for structured comparison.
  • Implemented batch processing to analyze multiple repositories from a CSV file, enabling large-scale repository evaluations.
  • Integrated GitHub API to automate repository analysis and refactoring based on best practices for repository organization.

CloudPref: Model Evaluation for Cloud Providers [View Repository]

  • Adapted the METR Task Standard to evaluate model preferences for cloud service providers, tailored to data procurement and storage scenarios.
  • Created methods to elicit model preferences for cloud providers (AWS, GCP, Azure), focusing on performance optimization.
  • Designed ranking methods to balance cost-effectiveness and deployment efficiency among cloud service providers.

Triton GPU Optimization Tutorials [View Repository]

  • Developed a series of tutorials to optimize GPU operations using Triton, focusing on effecient inference and RLHF applications.
  • Topics include elementwise operations, kernel fusion, memory reuse, quantization, matrix multiplication optimization, and RLHF tuning.
  • Each tutorial benchmarks Triton’s performance against CUDA to develop custom kernels for optimized GPU computations.
